VBMan Editor

Edit Unity(C#) Script in Visual Basic .NET




Download v1.0.0

How to use?

Extract VBMan in your desired location. Open Unity and select Edit -> Preferences -> External Tools. Click on MonoDevelop and select browse and choose VBMan Editor.exe. From next, script will open in VBMan Editor
